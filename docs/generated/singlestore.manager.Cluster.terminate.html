singlestore.manager.Cluster.terminate¶

Cluster.terminate(wait_on_terminated: bool = False, wait_interval: int = 10, wait_timeout: int = 600) → None¶

Terminate the cluster.

Parameters

  • wait_on_terminated (bool, optional) – Wait for the cluster to go into ‘Terminated’ mode before returning

  • wait_interval (int, optional) – Number of seconds between each server check

  • wait_timeout (int, optional) – Total number of seconds to check server before giving up

Raises

ClusterManagerError – If timeout is reached
