singlestore.manager.Cluster.update¶

Cluster.update(name: Optional[str] = None, admin_password: Optional[str] = None, expires_at: Optional[str] = None, size: Optional[str] = None, firewall_ranges: Optional[collections.abc.Sequence[str]] = None) → None¶

Update the cluster definition.

Parameters

  • name (str, optional) – Cluster name

  • admim_password (str, optional) – Admin password for the cluster

  • expires_at (str, optional) – Timestamp when the cluster expires

  • size (str, optional) – Cluster size in cluster size notation (S-00, S-1, etc.)

  • firewall_ranges (Sequence[str], optional) – List of allowed incoming IP addresses
