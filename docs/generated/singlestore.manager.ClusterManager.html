singlestore.manager.ClusterManager¶

class singlestore.manager.ClusterManager(access_token: Optional[str] = None, version: Optional[str] = None, base_url: Optional[str] = None)¶

SingleStore cluster manager.

This class should be instantiated using singlestore.manage_cluster().

Parameters

  • access_token (str, optional) – The API key or other access token for the cluster management API

  • version (str, optional) – Version of the API to use

  • base_url (str, optional) – Base URL of the cluster management API
